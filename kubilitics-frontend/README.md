# Kubilitics Frontend

Kubernetes dashboard and cluster management UI.

## Development

**Requirements:** Node.js & npm ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

```sh
npm install
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build

```sh
npm run build
```

## Test

```sh
npm run test
npm run test:e2e
```
