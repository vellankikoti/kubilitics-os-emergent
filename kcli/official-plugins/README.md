# Official kcli Plugins

Included plugins:
- cert-manager
- argocd
- istio
- backup

Install a plugin from this repo:

```bash
kcli plugin install ./official-plugins/cert-manager
kcli plugin install ./official-plugins/argocd
kcli plugin install ./official-plugins/istio
kcli plugin install ./official-plugins/backup
```

Install all official plugins:

```bash
./scripts/install-official-plugins.sh
```
